### 类型别名
类型别名用来给一个类型起个新名字

### 简单例子
```ts
type Name = string
type NameResolve = () => string
type NameOrResolve = Name | NameResolve

function getName(n: NameOrResolve): name {
    if (typeof n === 'string') {
        return n
    } else {
        return n()
    }
}
```
这个例子中我们使用type创建类的别名，类型的别名常用与联合类型
